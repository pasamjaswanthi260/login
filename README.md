# Login & Registration System

This is a beginner-level full stack login and registration web application.

## Technologies Used
- HTML
- CSS
- Python (Flask)
- SQLite

## Features
User can create an account
- User can login using username and password
- User stays logged in using session
- User can see a dashboard after login
- User can logout from the application
-

## Project Type
Beginner Full Stack Project (No APIs)

## How to Run the Project

pip install flask
